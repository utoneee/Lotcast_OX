# Lotcast OX — 是非吉凶

心中有疑，神明指引。一款東方廟宇風格的占卜小程式，單頁 HTML 即開即用。

> 此應用為 LINE 貼圖 [尖尖圓圓 2](https://store.line.me/stickershop/product/33580655/zh-Hant) 衍生創作。占卜結果僅供娛樂，請理性思考勿迷信。

## 模式

| 模式 | 說明 | 結果 |
|------|------|------|
| 是非神君 | 擲筊問是非 | 是 / 否 / 再一次 |
| 吉凶判官 | 抽一支籤問吉凶 | 大吉 / 吉 / 凶 / 大凶 |
| 批次抽籤 | 多人 / 多籤一次抽完 | 各自顯示結果 |

## 特色

- 🏮 暖灰紙感配色、明體字、香火粒子特效，廟宇氛圍
- 🎡 首頁神像輪播切換模式
- 📱 可「加入主畫面」當 App 用，支援離線快取（PWA）
- 🔗 一鍵複製分享連結
- 🪶 純 HTML + CSS + JS，無框架依賴

## 加入主畫面

iOS：在瀏覽器點分享鍵，選「加入主畫面」即可像 App 一樣全螢幕使用，並可啟用離線。

## 技術

- 單一 `index.html`，圖片以 WebP 提供高解析度
- Service Worker (`sw.js`) 處理離線快取與版本更新
- 設計尺寸 390 × 844 px（iPhone 14），手機採 width-only 縮放
